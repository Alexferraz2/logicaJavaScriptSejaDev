Selecionando elementos

pelo Id é document.getElementbyId('Escre a Id do elemento')
pela classe é document.getElementsbyclassname('Escre a classe do elemento')
pela tag é document.getElementsbyTagName('Escre a tag do elemento')
pela propriedade 'Name' é document.getElementsbyName('Escre a propriedade Name do elemento')

e ainda temos uma outra maneira de selecionar que é pelo document.queryselector(''), dentro dos parenteses colocamos as id, classe, tag e atributo Name, como informamos dentro do CSS, exemplo da seleção da id que temos no html: document.queryselector('#exemplo'), se for classe será assim document.queryselector('.lista'), mas dessa maneira ele irá pegar o conteúdo da primeira classe lista, para pegar todas as classes lista tem que ser feito dessa maneira document.queryselectorall('.lista')
