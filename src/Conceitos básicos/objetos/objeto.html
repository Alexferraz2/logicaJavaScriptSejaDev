Objeto

Objeto é uma listagem nomeada, diferente do array que é numerada.
e objeto é declarado um pouco diferente do array. veja abaixo as diferença.

Array

let carros = [ 'Palio' 'Uno' 'Corolla' 'Ferrari'];

Objeto

let carro = { nome: 'Fiat', modelo:'Uno', peso:'800kg' };

O array declaramos os valores entre [] e o objeto entre {}
