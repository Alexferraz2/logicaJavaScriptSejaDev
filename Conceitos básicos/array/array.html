Array

É uma variável que armazena vários valores diferente, ordenados numericamente
Ex: let carros = ['Palio', 'Uno', 'Corolla', 'Ferrari'];
nesse caso a váriavel carros, está armazenando quatros valores, e para transformar uma varíavel em uma array insere os valores entre cochetes e sepados um do outro por vírgula. para exibir o array no console.log faz dessa maneira console.log(carros); , e para motrar um valor específico faz dessa maneira console.log(carros [2]); , no caso ele irá mostrar o terceiro valor, pois o primeiro valor do array, é vinculado à ordem 0. E temos como montar um array dentro de uma array, veja abaixo o exemplo:
let ingredientes = [
['uva', 'pera', 'laranja'],
['arroz', 'feijão', 'macarrão']

];

console.log(ingredientes [0] [1]); , no console.log eu informei qual o array que deveria ser acessado e qual ítem a ser acessado, lembrando que a ordem começa do 0.
