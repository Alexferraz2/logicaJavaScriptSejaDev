queryselector

O queryselector ele serve para selecionar elementos, id ou classes que temos no html. O código se escreve assim document.querySelector(".lista"); , dessa maneira ele vai selecionar todas os elementos que tiverem a classe lista, para selecionar uma id faz dessa maneira document.querySelector('#titulo'); e para selecionar uma classe que está dentro de um elemento como uma ul ou ol, faz dessa maneira document.querySelector('ul.lista'); nesse caso caso ele vai selecionar apenas a classe lista que está dentro da ul. O queryselector, ele apenas vai selecionar o primeiro valor que está no html.
Para mudar o estilo de algum elemento faz dessa maneira document.querySelector(".lista").style.color = '#FF0000'; mas ele vai fazer isso no primeiro elemento que tiver a classe lista, o outro elemento que tiver a classe lista será ignorado.

querySelectorAll

O querySelectorAll ele retorna uma array, então ele pegará tudo que for informado para selecionar, e faz da seguinte maneira document.querySelectorAll('lista'). para selecionar selecionar um determinado ítem faremos dessa maneira document.querySelector("ul > li:nth-child(3)"); dessa maneira ele selecionou o terceiro ítem da lista que está dentro de uma ul.
Para mudar o estilo quando o seletor for queryselectorall, não podemos fazer como fizemos no queryselector, pois o queryselectorall, ele retorna um array, então teremos que fazer um for, para que o código percorra toda a array da classe, id ou elementos selecionado. Veja um exemplo abaixo:
1º cria uma variável para receber o que estamos selecionando para poder manipular, veja abaixo um exemplo:
let lista = document.querySelectorAll('li');
2º faz o for para que o código percorra todos os ítens do array
for(let i in lista) {
lista[i].style.color = '#ff0000';
}
